# Titanic Data Analysis using Pandas

## Dataset
This project uses the Titanic dataset, which contains information about passengers such as age, gender, passenger class, fare, and survival status.

## Data Cleaning
- Filled missing Age values with the mean.
- Filled missing Embarked values with the most frequent value.
- Replaced missing Cabin values with "Unknown".

## Analysis Performed
- Displayed the first five rows.
- Checked missing values.
- Generated summary statistics.
- Counted survived and non-survived passengers.
- Calculated average passenger age.
- Calculated average fare by passenger class.
- Analyzed survival rate by gender.

## Key Findings
- Female passengers had a higher survival rate than males.
- First-class passengers paid a higher average fare.
- Missing values were cleaned successfully.
- A cleaned dataset was saved as `cleaned_titanic.csv`.
